01 Section About

Clean architecture. Tight code. Systems built to last.

I've been writing Luau since 2018. The work I take on tends to sit at the harder end — physics simulations, IK rigs, server-authoritative combat, exploit-resistant data pipelines. The kind of system that has to be right the first time because rewriting it costs the project months.

I work inside a framework discipline (Managers / Controllers, packet-based networking, type annotations everywhere) and I keep code legible for the next person who reads it. Optimised, organised, and documented through structure rather than comment noise.

In 2024 I generated $30,000 from Roblox work — creating, investing, updating & selling game % to big companies. The games I've worked on have crossed 500 million visits and peaked at 75,000+ concurrent players across all experiences.
